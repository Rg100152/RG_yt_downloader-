🌐 R.G DARKWEB DOWNLOADER v2.0
R.G DarkWeb Downloader ek high-performance, GUI-based media downloader hai. Iska interface classic terminal aur matrix aesthetics se inspired hai, jo aapko ek "hacker" vibe deta hai jab aap apna favourite content download karte hain.

⚡ Main Features
Advanced Terminal UI: Blinking cursor, typewriter effects, aur dynamic console logs.

Smart Analysis: Download karne se pehle video ka title, uploader, aur views analyze karein.

High-Speed Downloads: yt-dlp engine ka use karke multi-threaded downloading.

Custom Quality Selector: 144p se lekar 4K tak (available formats ke basis par).

Audio Specialist: MP3 extraction @ 192kbps high quality.

Playlist Ninja: Poori playlist ko fetch aur download karne ki kshamta.
📥 Installation
1. Prerequisite (FFmpeg)
Ye tool videos ko merge karne ke liye FFmpeg ka use karta hai.

Windows: ffmpeg.org se download karein aur Environment Variables mein add karein.

Linux: sudo apt install ffmpeg
